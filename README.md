## üöÄ How to Use  

1.  **Install & Setup**:
    -   Click the extension icon in your toolbar.
    -   Enter your [OpenRouter API Key](https://openrouter.ai/).
    -   (Optional) Choose a specific model (default: `mistralai/mistral-7b-instruct:free`).

2.  **Activate Overlay**:
    -   Click any text box on any website.
    -   Press **Cmd+Shift+Y** (Mac) or **Ctrl+Shift+Y** (Win).

3.  **Manage Prompts**:
    -   **Add**: Click **+ Add New Prompt**. Type a name and text, then Save.
    -   **Right-Click Save**: Select any text on a page -> Right-Click -> **Save as Prompt**.
    -   **Optimize**: In the "Add" view, click the **‚ú®** button to let AI refine your text.
    -   **Inject**: Click any prompt in the list to insert it into the webpage.
